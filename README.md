## Usage

[Helm](https://helm.sh) must be installed to use the charts.
Please refer to Helm's [documentation](https://helm.sh/docs/) to get started.

Once Helm is set up properly, add the repository as follows:

```console
helm repo add https://github.com/nholuongut https://github.com/https://github.com/nholuongut/helm-charts
```

You can then run `helm search repo https://github.com/nholuongut` to see the charts.
